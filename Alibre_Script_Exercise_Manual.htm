Running a script to produce a bolt

2


1


  1. Copy and paste the Bolt Creator script into a New Script window

Editing the Bolt Parameters


In the Bolt Creator script, lines 5, 6, 10, 11, 14, and 15 all hold values associated with the

Bolt Head, Bolt Shoulder, and Hex parameters.


Manually change some of these values and re-run the script to see changes in the bolt output (*In the image above, the bolt shoulder extrusion length has been changed to a value of "40")


1


  1. Confirm results of your parameter changes to the Bolt Creator script (Example image above shows results of Bolt with Shoulder Extrusion parameter changed to 40).

Save and Export the Results




Lines 20-22 of the script provide options to Save the part file, Export the part file to STL

format, and Close the workspace after the bolt has been generated.


To do the above mentioned just remove the "comment out" marks (delete the "#" symbols) at the beginning of those lines, and edit the directory paths so that they reflect directory paths currently present on your computer. Then re-run the script.
